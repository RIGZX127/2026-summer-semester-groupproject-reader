# Team Division

## Member 1: Core Reading System

### Responsible For
- RSS订阅与解析
- 文章同步
- 内容清洗
- 基础阅读界面

### Task Goal
实现 RSS 阅读器核心功能，使用户可以：
- 添加RSS源
- 获取文章
- 查看文章内容

### Workflow

1. 使用 feedparser 完成RSS解析
2. 使用 BeautifulSoup 清理文章内容
3. 使用 Markdown 格式化文章
4. 使用 PyQt6 实现阅读界面

# Collaboration Rules

## Interface First
模块之间通过统一数据结构连接，避免直接依赖。

## Git Workflow
- 每人独立开发
- 定期提交代码
- 保留清晰提交记录

## Integration Order

Database  
↓  
RSS Reader  
↓  
AI Service  
↓  
Notes & Tags  
↓  
UI Optimization
